Guida rapida per sviluppatori
1. Ottenere un token:

curl -X POST https://db1.bisuite.app/api/v1/oauth/token \
-H "Content-Type: application/json" \
-d '{
    "grant_type": "client_credentials",
    "client_id": "IL_TUO_CLIENT_ID",
    "client_secret": "IL_TUO_CLIENT_SECRET"
}'
Risposta: {"access_token": "...", "refresh_token": "...", "expires_in": 3600, ...}

2. Usare il token nelle chiamate:

curl https://db1.bisuite.app/api/v1/sales?from_date=2026-01-01&limit=10 \
-H "Authorization: Bearer IL_TUO_ACCESS_TOKEN"
3. Rinnovare il token:

curl -X POST https://db1.bisuite.app/api/v1/oauth/token \
-H "Content-Type: application/json" \
-d '{
    "grant_type": "refresh_token",
    "refresh_token": "IL_TUO_REFRESH_TOKEN"
}'
4. Health check: GET https://db1.bisuite.app/api/v1/health (pubblico, nessun token richiesto)

5. Dashboard admin: https://db1.bisuite.app/dash
